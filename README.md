# Glyphs-Scripts

## Word Generator

**```Dicitonary.py```** utilizes the system dictionary to generate word strings using only characters that are available and drawn in the current master.

## Spacing

**```BasicSpacingString```** is a shortcut for generating uppercase spacing strings from A–Z. Running the script opens **two separate tabs** with sequences for uppercase and lowercase respectively.


**```ComplexSpacingString```** generates every possible combination of uppercase, lowercase, and uppercase–lowercase pairings (e.g. HHHABAHHH, HHHabaHHH, HHHAbAHHH) for thorough spacing evaluation in a consistent context.

| Basic   | Complex   |
| ------- | --------- |
| HHHAHHH | HHHAaAHHH |
| HHHBHHH | HHHAbAHHH |
| HHHCHHH | HHHAcAHHH |

## Glyphs

**```EmptyGlyphs```** Opens all empty glyphs (characters that does not contain paths or components) in current master.
